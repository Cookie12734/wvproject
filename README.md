冬休み課題でvalorantというゲームの募集掲示板を作成しました。
募集自体はX等でできますが、その人がどんな人なのか分かりづらく、不安要素があるため、それを排除するために他人からの評価を確認できる掲示板を
作成しようという思いでこの掲示板を作成しました。
主な機能としては、ログイン機能、募集機能、コメント機能、評価機能(good, bad)、通報機能、チャット機能、ブロック機能、通知機能、お知らせ機能があります。
ログイン機能は実際にメールが届く仕組みになっており、メールアドレスの変更やパスワードのリセットなどが行えます。
また、管理者権限を持つアカウントはユーザー管理(チャットの削除、募集の削除、BAN)を行うことができたり、通報を行うと、管理者にしか見えない
通報管理タブから通報の内容を見ることができます。
また、このwebアプリケーションはpythonanywhereへデプロイしています。
